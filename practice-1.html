符號 語法

  • 不斷行空白指的是, 被他隔開的前後兩字一定會在一起顯示, 不會被換行隔開, 適合用在數字連接單位的時候
  • 有一個不斷行連字符(non-breaking hyphen) entity number : &#8209;
  • 上面的指令是entity name, 但有可能瀏覽器顯示不出來, 可改用entity number
  • 更多奇怪符號、entity number : ref 1, ref 2

表格設定

  • { border-collapse: collapse; }可以讓表格的外框維持單線條
  • table{ border-spacing: 5px; }可以調整table跟cell框線之間的距離
  • padding對th, td有效, 對table似乎沒有效果
  • colspan/rowspan屬性用來擴張儲存格
  • 利用:nth-child() 抓取特定欄位
  • <colgroup>裡面放<col>, 用來訂出column群組, 寫在<caption>之後, <tr>之前 (example)
  • <thead>, <tfoot>, <tbody>

Horizontal List

  • Home
  • News
  • Contact
  • About
  • li { float: left; } 做出橫列效果, li { display: inline; } 也行
  • ul { overflow: hidden; }, 有這項設定才會讓ul的範圍展開(背景顏色範圍), 不太明白為什麼
  • 樣式設定大多寫在a上, 且必須寫 a { display:block; }, a的padding才會把li撐開

<address> element defines contact information (author/owner) of a document or an article

<cite> element defines the title of a work 

					<pre>段落內保留換行及空白格式,標籤仍然有效, 似乎不會繼承CSS樣式, 
					但換行後, 文字編輯器為了對齊而產生的空白也會被加進來, 而且內容可以超出div的寬度

<abbr>配上title屬性, WTF

<bdo>配上dir屬性, 改變文字輸出的方向 : This text will be written from right to left.

<!--[if IE 9]> conditional comment, 符合條件的話不會被註解掉 <![endif]-->

Use the <map> tag to define an image-map. An image-map is an image with clickable areas. 用name屬性來跟圖片連結, 但定位方式是絕對座標, 圖片大小一改變, 位置就跑掉了

<noscript> 給不支援javascript的瀏覽器顯示的替代文字
